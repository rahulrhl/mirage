# Mirage

A dashboard web app.

## How To Use

### Prerequisites

You need NodeJS (6.0 or higher) with NPM.

### Build & Run

1. Install dependencies with `npm install`.
1. Run with `npm run app`.
1. Go to localhost:3000 to view it.
